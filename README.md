# Exploratory Data Analysis (EDA)

## Project Title
Exploratory Data Analysis on Titanic Dataset

## Objective
To explore the Titanic dataset, identify trends, patterns, and data issues using Python.

## Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib

## Dataset
Titanic Dataset (titanic.csv)

## Analysis Performed
- Loaded the dataset
- Displayed first 5 rows
- Checked data types
- Checked missing values
- Generated summary statistics
- Created Bar Chart
- Created Pie Chart
- Created Histogram

## Conclusion
Successfully performed Exploratory Data Analysis (EDA) and identified trends, patterns, and missing values.
